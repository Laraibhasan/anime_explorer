# Anime Explorer 🎌

Anime Explorer is a Netflix-style web app to search and explore anime using the Jikan API. Users can view top-rated anime, filter by genre, and load more results dynamically — all in a sleek dark UI.

## Features
- 🔍 Search anime with autocomplete
- 🎭 Filter by genre (no reload)
- 🎞️ View score, episodes, and synopsis in modal
- 📥 Load more anime without page reload
- 🎨 Stylish UI using black and red palette

## Tech Stack
- HTML, CSS, Vanilla JS
- Node.js, Express, EJS
- Jikan API (MyAnimeList)

npm install
node server.js
